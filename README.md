# MasterTime  
- Веб приложение в котором есть своя база данных, два пользователя, интеграция swagger.
- Чтобы запустить проект необходимо загрузить пакеты (команда: npm i ), затем запустить сервер (команда: npm start).
# Используемые зависимости
- bcrypt - для хеша паролей
- dotenv - для хранения переменных
- ejs - позволяет использовать динамические данные в HTML
- express - для создания сервера
- express-session - для хранения данных о пользователе.
- mysql2 - подключение к базе данных
- sequelize - для работы с базой данных через js
- swagger-ui-express - документация swagger через браузер
